# Сердце практики Дхаммы

Конспект по раннесуттовой Дхамме: карта, воркбук и словарь палийских терминов.
Актуальная версия находится в ветке main. Изменения можно смотреть в истории комитов.

Документ содержит основные темы практики: четыре `ariyasacca`, восьмеричный путь, пять предписаний, `sīla`, `sati`, `samādhi`, джханы, `paṭiccasamuppāda`, `anattā`, `dukkha`, `taṇhā`, `upādāna`, `bhava`, пять `khandhā`, шесть сфер опыта и практические способы распознавания цепляния в повседневной жизни.

Этот текст не претендует на окончательность или безошибочность, является рабочей картой для сверки, практики и дальнейшего уточнения.
Не является медицинской, психологической или психотерапевтической рекомендацией.

В конце находятся словарь и ссылки на все источники, которые использовались при формировании.
